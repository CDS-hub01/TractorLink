# TractorLink 🚜

**TractorLink** is a web platform that connects farmers with tractor owners for efficient equipment hiring. The app simplifies booking, registration, and management of tractor rental services.

### Features
- **User Registration**: Farmers can create accounts to access tractor services
- **Hire Page**: Browse and request available tractors based on location + availability  
- **Admin Dashboard**: Manage bookings, users, and tractor listings from one panel
- **Responsive Design**: Works smoothly on desktop and mobile devices

### Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript ES6
- **Styling**: Custom CSS with responsive layout
- **No Frameworks**: Built with vanilla JS for learning + performance

### Folder Structure
